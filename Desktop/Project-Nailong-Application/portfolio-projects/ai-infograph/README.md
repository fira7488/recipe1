# AI Evolution Infographic

A polished, interactive one-page infographic that visualizes the evolution of artificial intelligence.
This frontend project combines responsive layout, animated timeline cards, data-driven counters, progress bars, and custom motion effects to create an immersive AI-themed landing page.

## Demo
- Open `index.html` in your browser.
- The page is designed for both desktop and mobile screens.

## Features
- Fixed navigation with smooth scrolling to sections
- Animated timeline cards for AI milestones
- Animated statistic counters and progress bars
- Popular AI tools showcase section
- Future-focused AI trend cards
- Particle background and cursor glow effects
- Interactive card hover lighting
- Scroll progress indicator

## Built With
- HTML5
- CSS3
- Vanilla JavaScript

## Project Structure
- `index.html` — markup and section structure
- `style.css` — layout, dark theme styling, animations, responsive design
- `script.js` — scroll progress, reveal animations, particle effects, counter updates, interactive cursor

## Usage
1. Clone or copy the project folder.
2. Open `index.html` in a web browser.
3. For local development, you can also serve the folder with a simple static server.

## Customization
- Update timeline events inside `index.html`.
- Edit the `data-target` values in the stats section to change counter output.
- Change the AI tool cards and future cards to reflect your own content.
- Adjust the CSS theme colors in `style.css`.

## Notes
- The page uses Google Fonts via CDN for typography.
- No JavaScript frameworks or build tools are required.
